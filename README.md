# Dono da Oficina — Site Institucional

Site de vendas e landing page para o sistema **Dono da Oficina** — sistema completo de gestão para oficinas mecânicas e centros automotivos.

## 🚀 Stack

- HTML5, CSS3 e JavaScript puro (vanilla)
- Zero dependências — máxima performance no PageSpeed
- Responsivo: mobile, tablet e desktop

## 📂 Estrutura

```
sitedonodaoficina/
├── index.html       → Página principal (single-page)
├── css/
│   └── styles.css   → Design system e estilos
├── js/
│   └── main.js      → Interações e animações
├── images/          → Screenshots do sistema (adicionar manualmente)
├── robots.txt       → SEO
└── README.md
```
